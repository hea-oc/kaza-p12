# Kaza - P12

Application immobilière avec un frontend Next.js et un backend Express.

## Prérequis

- Node.js
- npm

## Installation

```bash
# À la racine du projet
npm install

# Dans le frontend
cd frontend && npm install

# Dans le backend
cd back/dev-react-P12 && npm install
```

## Lancer l'application

À la racine du projet :

```bash
npm run dev
```

Cela démarre en parallèle :
- Le frontend sur [http://localhost:3000](http://localhost:3000)
- Le backend sur [http://localhost:3001](http://localhost:3001)

## Structure

```
kaza_project/
├── frontend/        # Next.js + Tailwind CSS
└── back/
    └── dev-react-P12/  # Express + SQLite
```
